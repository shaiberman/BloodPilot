This code requires:
python>=3.6
pip install flask
pip install pandas

You then need to run "python app.py" to run a local web server.
To access the example study in this app, you can go to http://127.0.0.1:5000/

Then- we will run the study on a global web server, through pythonanywhere.

Tutorials and general guidelines can be found in:
https://github.com/alicexue/MTurk/blob/master/MTurk_Web_App_Guide.pdf
https://alicexue.gitbook.io/online-study-tutorial/
